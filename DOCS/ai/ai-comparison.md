
# Comparativa de Asistentes de IA
En este documento comparo ChatGPT, Claude y Gemini.
Conceptos Técnicos
- ChatGPT:** Muy bueno explicando el DOM con ejemplos sencillos.
- Gemini:** Fue el más útil para entender errores de despliegue en Vercel (me explicó lo de Jinja2).
- Claude:** Genera código muy limpio para JavaScript.

### Detección de errores
Al probar funciones con bugs, Gemini detectó rápido que el problema del requirements.txt era el formato del archivo (BOM), mientras que otros sugerían errores de código.
